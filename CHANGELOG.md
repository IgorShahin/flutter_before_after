# Changelog

## 1.0.0

- Initial release.
- Added `BeforeAfterImage` and `BeforeAfterLayout`.
- Added draggable divider with custom overlay styling.
- Added pinch zoom, pan, and double-tap reset with `ZoomController`.
- Added `fixedLabels` option to control label behavior during zoom/pan.
- Improved slider interaction to allow drag from the full divider line area.
